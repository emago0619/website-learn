このページで学ぶこと

このページでは、Webサイトを作成・修正するために必要な3つのツールを準備します。 それぞれのツールの役割と、インストール方法を順番に説明します。

  • GitHub：コードを保存・共有するためのサービス
  • Cursor：AI搭載のコードエディタ
  • CodexCLI：コマンドラインで操作するためのツール

1. GitHubアカウントの作成

GitHubは、コードを保存・管理するためのサービスです。 無料でアカウントを作成でき、Webサイトのファイルを保存したり、変更履歴を管理したりできます。

手順

  1. GitHubの公式サイトにアクセス
    ブラウザで https://github.com を開きます。
  2. アカウント作成
    「Sign up」ボタンをクリックし、以下の情報を入力します：
    • ユーザー名（英数字とハイフンのみ）
    • メールアドレス
    • パスワード（8文字以上、英数字と記号を含む）
  3. メール認証
    登録したメールアドレスに認証メールが届きます。 メール内のリンクをクリックして認証を完了させましょう。
  4. 初期設定
    初回ログイン時、いくつかの質問に答えますが、後で変更できるので、まずは「Skip」でスキップしても問題ありません。
完了！ GitHubアカウントの作成が完了しました。 次はCursorのインストールに進みましょう。

2. Cursorのインストール

Cursorは、AI機能が搭載されたコードエディタです。 通常のエディタと違い、AIのサポートを受けながらコードを書くことができます。 非エンジニアの方でも、AIの助けを借りて簡単にWebサイトを作成・修正できます。

手順

  1. Cursorの公式サイトにアクセス
    ブラウザで https://cursor.sh を開きます。
  2. ダウンロード
    「Download」ボタンをクリックし、お使いのOS（Windows、Mac、Linux）に合わせてダウンロードします。
  3. インストール
    ダウンロードしたファイルを実行し、インストールウィザードに従ってインストールします。
    • Windows： ダウンロードした.exeファイルをダブルクリック
    • Mac： ダウンロードした.dmgファイルを開き、アプリケーションフォルダにドラッグ
    • Linux： ダウンロードしたファイルを解凍し、適切な場所に配置
  4. 初回起動
    Cursorを起動すると、初期設定画面が表示されます。 必要に応じて設定を変更できますが、デフォルトのままで問題ありません。
  5. GitHubアカウントとの連携（推奨）
    Cursorの設定から、GitHubアカウントと連携すると、より便利に使えます。 設定 > Accounts > GitHub から連携できます。
ヒント： Cursorは無料で使用できますが、高度なAI機能を使う場合は有料プランが必要な場合があります。 基本的な機能は無料で使えるので、まずは無料版で試してみましょう。
完了！ Cursorのインストールが完了しました。 次はCodexCLIのインストールに進みましょう。

3. CodexCLIのインストール

CodexCLIは、コマンドライン（ターミナル）からGitHubやGitの操作を行うためのツールです。 初心者の方には少し難しいかもしれませんが、このガイドに従えば問題なく使えるようになります。

CodexCLIとは

CodexCLIは、GitHubの操作を簡単にするためのコマンドラインツールです。 ファイルのアップロード、変更の保存、Webサイトの公開など、多くの操作をコマンドで実行できます。

手順（Mac / Linuxの場合）

  1. ターミナルを開く
    Macの場合は「ターミナル」アプリ、Linuxの場合は「Terminal」を開きます。
  2. CodexCLIをインストール
    ターミナルに以下のコマンドを入力してEnterキーを押します： 
    curl -fsSL https://codexcli.com/install.sh | sh
  3. インストール確認
    以下のコマンドでインストールが成功したか確認します： 
    codex --version
    バージョン番号が表示されれば成功です。
  4. GitHubアカウントとの連携
    以下のコマンドでGitHubアカウントと連携します： 
    codex auth login
    ブラウザが開くので、GitHubアカウントでログインして認証を完了させます。

手順（Windowsの場合）

  1. PowerShellを開く
    Windowsキーを押して「PowerShell」と入力し、PowerShellを開きます。
  2. CodexCLIをインストール
    PowerShellに以下のコマンドを入力してEnterキーを押します： 
    iwr -useb https://codexcli.com/install.ps1 | iex
  3. インストール確認
    以下のコマンドでインストールが成功したか確認します： 
    codex --version
    バージョン番号が表示されれば成功です。
  4. GitHubアカウントとの連携
    以下のコマンドでGitHubアカウントと連携します： 
    codex auth login
    ブラウザが開くので、GitHubアカウントでログインして認証を完了させます。
注意： CodexCLIのインストールでエラーが出る場合は、 FAQページのトラブルシューティングセクションを確認してください。
完了！ CodexCLIのインストールが完了しました。 これで準備は完了です。次は基礎知識編に進みましょう。

4. インストール確認

すべてのツールが正しくインストールされているか確認しましょう。

確認チェックリスト

  • □ GitHubアカウントにログインできる
  • □ Cursorが起動できる
  • □ CodexCLIがインストールされている（codex --versionで確認）
  • □ CodexCLIとGitHubが連携されている（codex auth statusで確認）
