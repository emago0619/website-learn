# 非エンジニア向け Webサイト構築ガイド

GitHub、Cursor、CodexCLIを使って、非エンジニアの方でもWebサイトの構築や修正ができるようになるための包括的なガイドサイトです。

## 概要

このプロジェクトは、プログラミングの経験がない方でも、以下の3つのツールを組み合わせてWebサイトを作成・修正できるようになることを目的としています：

- **GitHub**：コードを保存・管理するサービス
- **Cursor**：AI機能が搭載されたコードエディタ
- **CodexCLI**：コマンドラインからGitHubを操作するツール

## サイト構成

このガイドサイトは、以下のページで構成されています：

1. **ホーム（index.html）**
   - ガイドの概要と目的
   - 学習の進め方
   - 各セクションへのナビゲーション

2. **準備編（pages/setup.html）**
   - GitHubアカウントの作成方法
   - Cursorのインストール手順
   - CodexCLIのインストール手順
   - 初期設定とアカウント連携

3. **基礎知識編（pages/basics.html）**
   - GitとGitHubの基本概念
   - Cursorの特徴と使い方
   - CodexCLIの役割と基本コマンド
   - 基本的な用語集

4. **実践編（pages/practice.html）**
   - 新規Webサイトの作成手順
   - GitHub Pagesでの公開方法
   - 既存Webサイトの修正方法
   - より高度な機能の追加方法

5. **FAQ（pages/faq.html）**
   - よくある質問と回答
   - トラブルシューティング
   - エラーメッセージの対処法
   - 参考リンク集

## 使い方

1. **ローカルで確認する場合**
   - `index.html`をブラウザで開く
   - または、ローカルサーバーを起動して確認する

2. **GitHub Pagesで公開する場合**
   - このリポジトリをGitHubにプッシュ
   - リポジトリのSettings > Pagesから公開設定を行う
   - 数分待つと、公開URLが表示される

## 技術仕様

- **HTML5**：セマンティックなマークアップ
- **CSS3**：モダンなスタイリングとレスポンシブデザイン
- **純粋な静的サイト**：JavaScriptは最小限（将来的に拡張可能）

## 対象読者

- プログラミングの経験がない、または初心者の方
- Webサイトを作成・修正したいが、どこから始めればいいかわからない方
- GitHubやCursor、CodexCLIについて知りたい方
- 既存のWebサイトを修正したい方
