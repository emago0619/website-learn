サンプルプロジェクト要件定義集

サンプルプロジェクトで紹介している各テーマごとに、完成ラインの目安となる要件定義を個別ドキュメントに分割しました。興味のあるテーマを選び、上から順に要件を満たしていくことで、学習用のサンプルプロジェクトを短期間で立ち上げられます。

目次

静的サイトプロジェクト

  • ポートフォリオサイト
  • ブログサイト
  • ランディングページ

動的サイトプロジェクト（React/Next.js）

  • コーポレートサイト
  • タスク管理アプリ

このドキュメントの使い方

  1. 作りたいプロジェクトのドキュメントを開き、目的や必須要件を確認します。
  2. 「必須要件」を完了ラインと定義し、公開や社内共有ができる状態を目指します。
  3. 「任意要件」は余裕があるときに追加して発展課題として活用します。
  4. READMEやNotionなどのプロジェクト管理ツールに該当ドキュメントのリンクを貼り、進捗管理に活用します。
