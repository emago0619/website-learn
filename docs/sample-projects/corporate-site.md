# コーポレートサイト要件定義（Next.js）

## プロジェクトの概要
- **目的**: Next.jsを使って更新しやすく表示速度の速い企業サイトを構築する。
- **主な利用者**: 企業のWeb担当者、Next.jsの学習者。
- **完成イメージ**: トップ／サービス／お知らせ／会社情報／採用／お問い合わせで構成される多ページサイト。

## 必須要件
1. 静的生成（SSG）またはISRによる高速表示
2. お知らせ一覧・詳細ページ（ヘッドレスCMSと連携できる設計）
3. 採用情報一覧（職種・勤務地・雇用形態のタグ表示）
4. お問い合わせフォーム（送信後のサンクスページとメール通知）
5. 多言語対応（日英切り替え。`next-intl`などを活用）
6. アクセシビリティに配慮したナビゲーション（キーボード操作・ARIA対応）

## 任意要件（余力があれば）
- サイト内検索（AlgoliaなどのSaaS連携）
- 事例紹介ページ（CMSの別コンテンツタイプ）
- サイトパフォーマンス計測（Lighthouseレポート、Core Web Vitals）

## UI/UXのポイント
- ブランドカラーをもとにしたデザインシステム（ボタンやカードなどの共通コンポーネント）
- PC・スマホ共通でヘッダー、フッター、パンくずリストを表示
- 固定ヘッダーを採用し、スクロール時は影を付与して視認性を高める

## 技術スタック
- Next.js 14
- TypeScript
- Tailwind CSS
- `next-seo`
- フォーム送信用: `react-hook-form` + `@tanstack/react-query`
