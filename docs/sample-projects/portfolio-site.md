# ポートフォリオサイト要件定義

## プロジェクトの概要
- **目的**: 個人のスキルや実績を整理し、採用担当者やクライアントに魅力を伝える。
- **主な利用者**: 就職・転職活動中のエンジニア・デザイナー、フリーランスのクリエイター。
- **完成イメージ**: ヒーロー、スキル、実績、連絡先の4セクションで構成されたシングルページの静的サイト。

## 必須要件
1. 自己紹介セクション（プロフィール、肩書き、SNSリンク）
2. スキル一覧（カテゴリーごとにアイコンやバッジ表示）
3. 代表的な制作実績3件以上（画像・説明・詳細ページへの導線）
4. 連絡手段（メールフォーム、またはメールアドレスの記載）
5. スマホでも崩れないレスポンシブ対応
6. ダークモード切り替えスイッチ

## 任意要件（余力があれば）
- ブログ記事や最新ニュースの掲載
- PDF形式の職務経歴書ダウンロード
- Google Analyticsによるアクセス解析

## UI/UXのポイント
- 落ち着いたメインカラーとアクセントカラーの2色構成
- セクション間に十分な余白をとり、スクロールで読みやすくする
- ダークモード時は背景色と文字色のコントラストを確保する

## 技術スタック
- HTML
- CSS（またはTailwind CSSなどのユーティリティ系CSS）
- 軽量なJavaScript

## 成果物チェックリスト
- `index.html`
- `/assets`配下の画像素材
- `styles.css`
- 環境構築・カスタマイズ手順をまとめたREADME
