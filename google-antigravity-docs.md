## 概要

Google Antigravityは、2025年11月18日にGoogleがGemini 3と同時にリリースした**エージェント型開発プラットフォーム**です。従来のAI IDE体験を維持しながら、エージェントファーストの未来に向けて進化したプラットフォームで、CursorなどのAI開発ツールに対抗する製品として位置付けられています。

### コアコンセプト

- **エージェントファースト・アーキテクチャ**: 開発者がタスク指向の高いレベルで作業できるよう、バックグラウンドエージェントがバックログタスクを処理し、コードベースのリサーチを行い、コンテキストスイッチングを削減
- **ミッションコントロール**: Agent Managerが複数のワークスペースでエージェントを生成、オーケストレーション、監視するための新しいコントロールサーフェスを提供
- **検証可能なワークフロー**: エージェントが自身の作業を自律的に検証し、信頼性の高いコードを生成

---

## 主要機能

### Agent Manager

Agent Managerは、Google Antigravityの中核となる機能です。

- **アクセス方法**: Editorのトップバーのボタンまたはキーボードショートカット `Cmd + E` から開く
- **専用ワークスペース**: エージェントに独立した専用ワークスペースを提供
- **複数エージェント管理**: 複数のワークスペースで複数のエージェントを作成、オーケストレーション、監視
- **非同期操作**: エージェントがバックグラウンドで自律的にタスクを実行

### 3つの作業サーフェス

エージェントはバックグラウンドで3つのサーフェスを並行して操作します：

1. **Editor（エディタ）**: コードの作成と編集
2. **Terminal（ターミナル）**: コマンドの実行とテストの実施
3. **Browser（ブラウザ）**: UIの検証と動作確認

### Artifacts（成果物）による検証

Antigravityのエージェントは「Artifacts」を生成します。これは生のツールコールよりもユーザーが検証しやすい形式の成果物です：

- **タスクリスト**: 実行するタスクの一覧
- **実装計画**: リサーチ後、実装前の計画書
- **スクリーンショット**: 作業結果の視覚的証拠
- **ブラウザ録画**: 機能の動作を示す録画
- **ウォークスルー**: 完成時の機能説明

#### 検証の重要性

> "エージェントが作業そのものだけでなく、その検証についても徹底的に考えることに重点が置かれています"

エージェントは：
- 結果をスクリーンショットで記録
- 機能が仕様通りに動作することを確認
- 問題を発見した場合は自動的にデバッグと修正を繰り返す

### フィードバックシステム

- ローカル操作で直感的な非同期ユーザーフィードバック
- テキストArtifactsへのGoogleドキュメントスタイルのコメント
- スクリーンショットへの選択＆コメントフィードバック

## システム要件

### macOS
- **最小バージョン**: macOS 12 (Monterey)
- **サポート**: Appleのセキュリティアップデートサポート対象バージョン（通常、現行と過去2バージョン）
- **注意**: X86はサポート対象外（Apple Siliconのみ）

### Windows
- **最小要件**: Windows 10 (64 bit)
- **ダウンロード形式**:
  - x64用
  - ARM64用

### Linux
- **要件**:
  - glibc >= 2.28
  - glibcxx >= 3.4.25
- **対応ディストリビューション例**:
  - Ubuntu 20以降
  - Debian 10以降
  - Fedora 36以降
  - RHEL 8以降
- **インストール方法**:
  - apt（Debianベース）
  - dnf（RPMベース）

## はじめに（Getting Started）

### ステップ1: ダウンロード

https://antigravity.google/download からインストーラーをダウンロード

### ステップ2: 認証

Googleアカウントで認証
- 寛大なレートリミットが付与
- Googleによると「5時間あたりのレートリミットに達するのはごく少数のパワーユーザーのみ」

### ステップ3: プロジェクト開始

Antigravityを開いて通常通りコーディング開始。3つのメインエリアが表示されます：
- **Editor**: コードエディタ
- **Agent Panel**: エージェントとの対話パネル
- **Manager**: Agent Manager

### クラウドネイティブオプション

ブラウザを開き、Googleアカウントで認証するだけで即座にコーディング開始可能
- 複雑なインストールや設定不要
- 数分で生産性を発揮

## ユースケース

### フロントエンド開発

**自律的なUI構築**
- 複雑なUIコンポーネントを説明するだけでエージェントがビルド、スタイリング、テストを実行
- CSSの微調整ではなくデザインビジョンに集中可能

**自動ワークフロー**
1. 新しいフロントエンド機能のコードを作成
2. ターミナルでlocalhostを起動
3. ブラウザで新機能の動作をテスト

### Webアプリ開発

**デモ例: フライト検索アプリ**
1. ユーザーが提供するフライト番号に基づいてフライトデータを返すWebアプリの作成を指示
2. ツールが実装計画を作成
3. コードを作成
4. Chromeブラウザを開いてアプリをテスト
5. テストのスクリーンショットをユーザーに提供

**パフォーマンス**
- 典型的なNext.js + Supabaseバックエンド機能を42秒で完成（Cursorは68秒）

### コードレビューと検証

**マルチエージェント並列処理**
- 1つのエージェントがリファクタリングを担当
- 別のエージェントがパフォーマンスを最適化
- 3つ目のエージェントがセキュリティコンプライアンスを確保
- シニアエンジニアリングチームがリアルタイムでPRをレビューするように協調

### 複数ファイルにまたがるバグ修正

データベーススキーマ、APIエンドポイント、フロントエンドコンポーネントに影響するバグを発見した場合：
- 従来のデバッグ: ファイルとツール間のコンテキストスイッチング
- Antigravity: エージェントがすべてを同時に追跡

### インタラクティブWebアプリ生成

Google AI Studioで、Gemini 3 Proは単一のプロンプトから完全にインタラクティブなWebアプリケーションを生成可能
- HTML、CSS、JavaScriptの複雑なオーケストレーションをバックグラウンドで処理

## アーキテクチャと技術詳細

### ブラウザコントロール

- Gemini Computer Useモデルとの密接な連携
- リアルタイムでドキュメントのリサーチやWebアプリのデバッグが可能

### 2つの分離されたビュー

1. **Editor**: コーディング作業
2. **Manager**: エージェント管理

これら2つのビューは意図的に単一のビューにグループ化されず、簡単に切り替えられるよう別々のスペースとして設計されています。

## まとめ

Google Antigravityは、従来のAI IDE体験を維持しながら、エージェントを専用サーフェスに昇格させ、Editor、Terminal、Browserへの直接アクセスを提供することで、AIアシスタンスを開発者のツールキットの一部からアクティブなパートナーへと変革します。

エージェントは複雑なエンドツーエンドのソフトウェアタスクを自律的に計画・実行し、自身のコードを検証できます。
